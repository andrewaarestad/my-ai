# My AI - Web Frontend

Next.js application for managing AI agents, MCPs, and authorizations.

## Getting Started

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

### Build

```bash
pnpm build
```

### Type Checking

```bash
pnpm type-check
```

### Linting

```bash
# Check for issues
pnpm lint

# Auto-fix issues
pnpm lint:fix
```

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Bundler**: Turbopack
